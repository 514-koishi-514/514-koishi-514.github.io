# 514-koishi-514.github.io

Personal GitHub Pages site for notes, projects, and format-friendly writing.

## What is inside

- Smooth animated static site with no build step
- Markdown and Quarto Markdown (`.qmd`) preview workspace
- Drag-and-drop local preview for `.md`, `.markdown`, `.qmd`, and `.txt`
- Example content under `content/`

GitHub Pages can serve this repository directly from the `main` branch.
